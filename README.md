# Online Resume Builder

## 📌 Project Description

The **Online Resume Builder** is a web-based application that helps users create professional and ATS-friendly resumes easily. It provides a simple interface where users can enter their details and generate a structured resume instantly.

## 🚀 Features

* Create resumes quickly and easily
* ATS score checking system
* Smart suggestions for improving resume content
* Multiple resume templates
* User-friendly interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
